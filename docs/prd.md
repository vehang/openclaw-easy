# 公众号排版工具 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位
一个面向公众号运营者的 Markdown 排版工具，支持多终端预览、多风格切换、富文本智能转换。

### 1.2 目标用户
- 公众号运营者
- 内容创作者
- 技术博主

### 1.3 核心价值
- **效率提升**：Markdown 一键转换公众号样式
- **多端预览**：实时预览手机/电脑端效果
- **智能转换**：飞书/公众号内容自动转 Markdown
- **一键提取**：URL 链接直接提取内容

## 3. 技术方案

### 3.1 技术栈
```
前端框架：React 18 + TypeScript
状态管理：Zustand
样式方案：Tailwind CSS + CSS Variables
MD 解析：markdown-it + highlight.js
富文本转换：turndown.js
编辑器：CodeMirror 6 / Monaco Editor
构建工具：Vite
```

### 3.2 项目结构
```
eh-mp-formatter/
├── src/
│   ├── components/         # 组件
│   │   ├── Editor/         # Markdown 编辑器
│   │   ├── Preview/        # 预览区域
│   │   ├── Toolbar/        # 工具栏
│   │   ├── ThemePicker/    # 主题选择器
│   │   ├── DevicePreview/  # 设备预览切换
│   │   └── UrlExtractor/   # URL 提取器
│   ├── hooks/              # 自定义 Hooks
│   ├── stores/             # Zustand 状态
│   ├── utils/              # 工具函数
│   │   ├── markdown.ts     # MD 解析
│   │   ├── converter.ts    # 富文本转换
│   │   ├── extractor.ts    # URL 提取
│   │   └── storage.ts      # 本地存储
│   ├── themes/             # 主题配置
│   ├── styles/             # 全局样式
│   └── App.tsx
├── public/
├── index.html
└── package.json
```

### 3.3 关键流程

#### Markdown 转换流程
```
用户输入 MD → markdown-it 解析 → 主题样式应用 → 内联 CSS 生成 → 复制到剪贴板
```

#### 富文本转换流程
```
用户粘贴 → 检测来源 → HTML 解析 → turndown 转换 → Markdown 输出
```

#### URL 提取流程
```
用户输入 URL → 后端代理请求 → HTML 解析 → 内容提取 → Markdown 输出
```

## 7. 风险与应对

| 风险 | 影响 | 应对方案 |
|------|------|----------|
| 公众号外链限制 | 链接无法点击 | 提示用户，提供脚注方案 |
| CORS 限制 | URL 提取失败 | 使用后端代理服务 |
| localStorage 容量 | 缓存失败 | 限制历史版本数量 |
| 浏览器兼容 | 部分功能异常 | 降级处理 |
