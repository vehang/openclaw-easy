### Docker 授权命令（重要！）

**问题**：容器重启后可能丢失 docker 组权限，导致无法操作 Docker

**解决命令**（在宿主机执行）：
```bash
docker exec -u root openclaw-easy-v3 sh -c 'echo "docker:x:993:node" >> /etc/group'
```

**触发条件**：容器重启后，如果执行 `docker ps` 提示 `permission denied`

**原因**：容器内 `/etc/group` 缺少 GID 993（docker 组）
