### Docker 相关问答

#### 1. docker-compose 转 docker run
用户有一个 docker-compose.yml，转换要点：
- `-d` 后台运行
- `--restart unless-stopped` 对应 `restart: unless-stopped`
- `-p` 端口映射
- `-v` 挂载卷
- `-e` 环境变量
- `build: .` 在 docker run 中无法使用，需先构建镜像

#### 2. 配置警告处理
```
Config warnings:
- plugins.entries.feishu-openclaw-plugin: plugin not found
```
这是无害的残留配置，系统自动忽略。要消除警告，删除配置文件中的对应条目即可。

#### 3. Docker 容器动态挂载 U 盘
- **不能**直接对运行中容器添加新挂载
- **解决方案**：预先创建占位目录并挂载到容器，U 盘插入后在宿主机上：
  ```bash
  # 方法一：直接挂载到占位目录
  mount /dev/sdb1 /data/usbshare

  # 方法二：绑定挂载
  mount --bind /media/your-usb /data/usbshare
  ```
- 容器内可立刻看到内容，无需重启

## 注意事项

- 容器重启后 Docker 权限可能丢失，需执行：
  ```bash
  docker exec -u root openclaw-easy-v3 sh -c 'echo "docker:x:993:node" >> /etc/group'
  ```
