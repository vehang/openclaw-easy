# TASK-003: 登录错误提示优化（浮动方案）

## 任务类型
代码优化（修复之前的问题）

## 问题描述
之前的方案用空白占位，导致没有错误时有一大块空白，体验不好。

## 新方案
使用浮动提示方式：
- 错误提示浮动在输入框附近
- 不占用布局空间
- 不影响其他元素
- 有错误时显示，没错误时完全隐藏

## 技术实现
- 使用 absolute 定位
- 或使用 Toast 组件
- 输入框边框变红作为辅助提示

## 验收标准
- [ ] 没有错误时无空白
- [ ] 有错误时布局不变
- [ ] 错误提示清晰可见
- [ ] 构建成功
- [ ] 部署验证
