# TASK-002: 登录错误提示优化

## 任务类型
代码优化 + Bug修复

## 需求描述
优化登录时的三种方式的错误提示，避免影响布局

## 问题清单

### 问题1: 账号密码输入错误提示影响布局
- 现象: 红字提示时隐时现，布局跳动
- 期望: 既能提醒用户，又不干扰布局

### 问题2: 验证码验证错误描述包含多余的 a 标签
- 现象: 后端返回错误包含 a 标签，不适合验证按钮场景
- 需要: 修改后端接口，区分不同场景的错误描述

### 问题3: 验证码错误提示影响布局
- 现象: 红字在输入框下方显示，影响布局
- 期望: 更舒适友好的提示方式

## 验收标准
- [ ] 错误提示不影响布局
- [ ] 后端接口错误描述正确
- [ ] 用户体验友好
- [ ] 构建成功
- [ ] 部署成功
- [ ] 测试验证通过
