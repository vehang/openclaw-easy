# 多机器人配置说明

## 当前架构

```
┌─────────────────────────────────────────────────────────────┐
│                    一个 OpenClaw 实例                        │
│                                                             │
│  飞书机器人 (会话隔离 ✅):                                    │
│  ┌─────────┐                                                │
│  │  小码   │──▶ 独立会话 → xiaoma/memory/                   │
│  └─────────┘                                                │
│  ┌─────────┐                                                │
│  │  犊子   │──▶ 独立会话 → duzi/memory/                     │
│  └─────────┘                                                │
│  ┌─────────┐                                                │
│  │  牛马   │──▶ 独立会话 → niuma/memory/                    │
│  └─────────┘                                                │
│                                                             │
│  共享规范:                                                   │
│  所有机器人读取 shared/SOUL.md                               │
└─────────────────────────────────────────────────────────────┘
```

---

## 已配置的隔离

### 会话隔离 (sessionIsolation: account)

| 机器人 | 会话 | 上下文 |
|--------|------|--------|
| 小码 | 独立 | 不与其他共享 |
| 犊子 | 独立 | 不与其他共享 |
| 牛马 | 独立 | 不与其他共享 |

### 记忆隔离 (memoryIsolation: account)

| 机器人 | 记忆目录 | 内容 |
|--------|----------|------|
| 小码 | xiaoma/memory/ | 独立记忆 |
| 犊子 | duzi/memory/ | 独立记忆 |
| 牛马 | niuma/memory/ | 独立记忆 |

---

## 共享规范

所有机器人读取同一份规范：

```
shared/
├── SOUL.md              # 能力规范
├── PRODUCT-DESIGN.md    # 产品设计规范
├── DEVELOPMENT.md       # 开发规范
└── DECISIONS.md         # 决策记录
```

---

## 配置详情

```json
{
  "channels": {
    "feishu": {
      "sessionIsolation": "account",
      "memoryIsolation": "account",
      "accounts": {
        "xiaoma": { "botName": "小码", ... },
        "duzi": { "botName": "犊子", ... },
        "niuma": { "botName": "牛马", ... }
      }
    }
  }
}
```

---

## 工作流程

### 收到消息时

```
1. 识别 account_id (xiaoma/duzi/niuma)
2. 加载对应会话上下文
3. 读取共享规范 shared/SOUL.md
4. 切换到对应工作目录 {account_id}/
5. 按规范执行任务
6. 记忆保存到 {account_id}/memory/
```

---

## 验证隔离

可以通过以下方式验证：

1. **会话隔离**: 在小码的对话中提到的内容，犊子和牛马不知道
2. **记忆隔离**: 小码的记忆只存在 xiaoma/memory/
