## 四、微信插件分析

### openclaw-easy 微信插件安装流程
1. **openclaw-base** — 安装基础插件到 `/root/.openclaw/extensions/`，mv 到 seed
2. **openclaw:root** — 安装 OpenClaw 本体，`openclaw plugins install` 时自带 openclaw-weixin
3. **openclaw:integrated** — 把 openclaw-weixin 从运行目录复制到 seed 目录

### 问题
- 镜像里的微信插件不是最新版，需要构建时用 CLI 更新
- CLI：`npx -y @tencent-weixin/openclaw-weixin-cli install`
- 最新版本：2.1.2（2026-03-31 发布）
- 版本列表：1.0.0 → 1.0.1 → 1.0.2 → 1.0.3 → 2.0.1 → 2.1.1 → 2.1.2
