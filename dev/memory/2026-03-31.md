## 问题记录

### npm workspace hoist 问题

workspace 根目录存在 `package.json` 时，npm 会把所有依赖 hoist 到根目录的 node_modules，导致子项目依赖找不到。

解决方案：移除根目录的 `package.json`，让每个项目独立管理依赖。

### npm 缓存权限问题

症状：`npm install` 成功但 `node_modules` 里没有对应的包。

解决方案：
```bash
sudo chown -R 1000:1000 /root/.npm
# 或使用临时缓存
npm install --cache /tmp/npm-cache-fresh
```
