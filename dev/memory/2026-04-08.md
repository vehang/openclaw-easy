# 2026-04-08 工作记录

## openclaw-easy 项目修复

### 1. Docker 构建版本号动态传递问题

**问题**：构建 `2026.3.13` 版本镜像，但启动后 `meta.lastTouchedVersion` 显示 `2026.4.1`

**原因**：
- `docker/init-integrated.sh` 中 heredoc 使用 `<< 'EOF'`（带引号），不解析变量
- 版本号硬编码为 `2026.4.1`
- Dockerfile 中 `ARG OPENCLAW_VERSION` 没有转为 `ENV`，运行时不可读

**修复**：
- `Dockerfile.integrated`: 添加 `ENV OPENCLAW_VERSION=${OPENCLAW_VERSION}`
- `init-integrated.sh`: `<< 'EOF'` 改为 `<< EOF`，版本号改为 `${OPENCLAW_VERSION}`

**提交**: `be2c186` on branch `root-version`

### 2. /api/config/simple 缓存持久化问题

**问题**：POST 设置后，GET 请求返回"暂无缓存配置"

**原因**：使用内存变量 `simpleConfigCache`，服务重启后丢失

**修复**：
- 添加缓存文件路径：`~/.openclaw/.simple-config.json`
- POST 时写入文件，GET 时读取文件
- 删除无用的内存变量
- 设置什么就返回什么，后续编辑 openclaw.json 不影响

**提交**: `be2c186` on branch `root-version`
