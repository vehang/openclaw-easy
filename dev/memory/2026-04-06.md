### 3. Gateway Token 自动生成

### 5. IM渠道验证逻辑优化

**问题：**
- 访问 `/api/config` 返回错误：`网易云信: App ID 不能为空; 网易云信: App Secret 不能为空`
- 用户只想配置其中一个IM渠道（飞书、QQ、企业微信、个人微信），但后端要求所有启用的渠道都有必填字段

**原因：**
- 后端 `validateAllChannels` 函数会验证所有启用的IM渠道
- 用户之前配置过网易云信(nim.enabled=true)，但没有填写appId和appSecret

**解决方案：**
修改验证逻辑：如果渠道启用了但缺少必填字段，自动禁用该渠道而不是报错

**提交：**
- `9e5524c` fix: IM渠道验证逻辑优化 - 启用但缺少必填字段时自动禁用而不是报错

### 8. 微信绑定状态保存方案

**问题：**
- 保存微信绑定状态时生成了 `channels.weixin`，导致 OpenClaw 报错：`unknown channel id: weixin`
- 微信是通过 openclaw-weixin 插件实现的，不应该是 channels 下的配置

**解决方案：**
1. 后端新增两个 API：
   - `GET /api/weixin/bound` - 获取微信绑定状态
   - `POST /api/weixin/bound` - 设置微信绑定状态
2. 绑定状态保存到 `$OPENCLAW_DIR/.weixin-bound` 文件
3. 移除 formToConfig 中对 `channels.weixin` 的处理

**提交：**
- `185e505` fix: 微信绑定状态保存到后端 .weixin-bound 文件，不再使用 localStorage
