### 14. 绑定状态数据结构

```json
{
  "channels": {
    "weixin": {
      "enabled": true,      // OpenClaw 标准字段
      "bound": true         // 自定义字段，标记绑定状态
    }
  }
}
```

**说明：** `bound` 是自定义字段，OpenClaw 会安全忽略，不影响正常使用。

### 15. 状态保存流程

```
登录成功
    │
    ▼
POST /api/config
{
    "channels": { "weixin": { "bound": true } }
}
    │
    ▼
保存到 openclaw.json
    │
    ▼
下次加载时从 GET /api/config 返回
    │
    ▼
populateConfig() 直接读取显示
```

### 9. 微信二维码展示功能

**问题：** ASCII 二维码在网页中显示错位

**解决方案：** 从控制台输出中提取二维码链接，转换为图片展示

**实现方式：**
1. 添加 qrcode.js 库
2. 监听 SSE 输出，正则匹配链接：`https?:\/\/liteapp\.weixin\.qq\.com\/q\/[^\s\]\n]+`
3. 使用 `QRCode.toCanvas()` 生成二维码图片
4. 页面布局：左侧 2/3 控制台，右侧 1/3 二维码

**关键代码：**
```javascript
// 提取二维码链接
const qrcodeMatch = data.text.match(/https?:\/\/liteapp\.weixin\.qq\.com\/q\/[^\s\]\n]+/);
if (qrcodeMatch) {
    updateWeixinQrcode(qrcodeMatch[0]);
}
```
