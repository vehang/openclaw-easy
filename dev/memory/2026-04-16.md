## 三、教训
- Docker 构建不要加 `--no-cache` 除非必要，会导致内存不足被 kill
- base 镜像构建完成后要验证工具链（`docker run which uv`），不要假设成功
- 不同项目的 Dockerfile 变更要做好版本管理
