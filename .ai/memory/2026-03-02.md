## Feishu 图片发送问题解决

### 问题
- 使用 message 工具发送图片时，`media` 参数不支持本地文件路径
- 用户只能看到路径字符串，而不是实际图片

### 解决方案
使用 `buffer` 参数发送 base64 编码的图片数据：

```javascript
// 1. 读取图片并转换为 base64
const base64Data = exec('base64 -w 0 /path/to/image.png');

// 2. 使用 buffer 参数发送
message({
  action: 'send',
  buffer: base64Data,  // base64 编码的图片数据
  mimeType: 'image/png',
  filename: 'image.png',
  message: '图片描述',
  target: 'user:xxx',
  channel: 'feishu'
});
```

### 关键点
- `media` 参数：只支持 URL，不支持本地路径
- `buffer` 参数：用于发送 base64 编码的附件
- 必须设置 `mimeType` 参数（如 `image/png`）
- 可以设置 `filename` 参数指定文件名
