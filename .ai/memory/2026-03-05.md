## --dangerously-skip-permissions 参数（重要发现）

### Claude Code 支持的权限模式
```bash
# 跳过所有确认框（推荐用于沙箱环境）
claude --dangerously-skip-permissions

# 允许手动选择跳过（不默认启用）
claude --allow-dangerously-skip-permissions

# 权限模式
claude --permission-mode <mode>
  - bypassPermissions: 跳过所有权限检查
  - acceptEdits: 自动接受编辑
  - dontAsk: 不询问
  - plan: 计划模式
  - default: 默认

# 打印模式（也跳过信任对话框）
claude -p, --print
```

### ⚠️ 注意事项
1. 首次使用需要手动确认 "Yes, I accept"
2. 仅在沙箱环境使用
3. 适合长时间无人值守任务
