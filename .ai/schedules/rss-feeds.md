## 工作流程

```
┌─────────────────────────────────────────────────┐
│              RSS Feed 获取流程                   │
├─────────────────────────────────────────────────┤
│                                                 │
│  1️⃣ 定时触发（8:00 / 12:00 / 20:00）            │
│                                                 │
│  2️⃣ 抓取 RSS Feed                               │
│  ┌─────────────────────────────────────────┐   │
│  │  36kr.com/feed                           │   │
│  │  huxiu.com/rss                           │   │
│  │  jiqizhixin.com/rss                      │   │
│  │  techcrunch.com/feed                     │   │
│  └─────────────────────────────────────────┘   │
│                                                 │
│  3️⃣ 筛选匹配关键词                              │
│  ┌─────────────────────────────────────────┐   │
│  │  关键词：AI、芯片、英伟达、大模型...        │   │
│  │  时间：最近6小时                          │   │
│  └─────────────────────────────────────────┘   │
│                                                 │
│  4️⃣ 抓取文章详情                                │
│  ┌─────────────────────────────────────────┐   │
│  │  web_fetch 获取全文                       │   │
│  │  提取关键信息                             │   │
│  └─────────────────────────────────────────┘   │
│                                                 │
│  5️⃣ 整理汇总                                    │
│  ┌─────────────────────────────────────────┐   │
│  │  分类整理                                 │   │
│  │  去重                                     │   │
│  │  生成卡片消息                             │   │
│  └─────────────────────────────────────────┘   │
│                                                 │
│  6️⃣ 推送                                        │
│  ┌─────────────────────────────────────────┐   │
│  │  通过 Notify Bot 发送                     │   │
│  └─────────────────────────────────────────┘   │
│                                                 │
└─────────────────────────────────────────────────┘
```

## RSS 解析示例

```xml
<!-- RSS Feed 结构 -->
<rss>
  <channel>
    <item>
      <title>英伟达发布新款GPU</title>
      <link>https://36kr.com/p/123456</link>
      <pubDate>2026-02-25 08:00:00</pubDate>
      <description>英伟达今日宣布...</description>
    </item>
  </channel>
</rss>
```
