## 商机筛选逻辑

### 1. 筛选条件
| 条件 | 说明 |
|------|------|
| ✅ 搜索量高 | 热度 > 100万 |
| ✅ 有商业价值 | 和消费、工具、服务相关 |
| ✅ 竞争度适中 | 不是巨头垄断 |
| ✅ 技术可行 | 独立开发者能做 |

### 2. 排除条件
| 条件 | 说明 |
|------|------|
| ❌ 娱乐八卦 | 明星、综艺 |
| ❌ 政治敏感 | 时政、社会事件 |
| ❌ 巨头市场 | BAT 已垄断 |

### 3. 商业分析维度
| 维度 | 说明 |
|------|------|
| 市场规模 | 有多少人需要 |
| 竞争程度 | 有多少竞品 |
| 技术门槛 | 开发难度 |
| 变现方式 | 怎么赚钱 |
| 差异化空间 | 能不能做出特色 |
