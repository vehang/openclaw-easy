## 注册指南

### Brave Search API（推荐优先）

1. 访问：https://brave.com/search/api/
2. 注册账户
3. 创建 API Key
4. 免费额度：2000次/月

### Google Custom Search

1. 访问：https://developers.google.com/custom-search
2. 创建搜索引擎
3. 获取 API Key 和 Search Engine ID
4. 免费额度：100次/天

### Bing Search API

1. 访问：https://azure.microsoft.com/services/cognitive-services/bing-web-search-api/
2. 创建 Azure 账户
3. 获取 API Key
4. 免费额度：1000次/月
